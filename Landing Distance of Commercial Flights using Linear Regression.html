Key Findings from the initial analysis are listed below:

  • Dataset FAA2 does not contain column duration which leads to 150 missing values in the combined dataset.
  • Variable speed_air contain 642 NA’s which can mislead the analysis/modelling if taken into consideration.
  • Variable height contains negative values which can be treated as abnormal.
  • Variable duration contains minimum value of 14.76 which is again abnormal in real life.
  • Variable distance contains minimum value of 34.08 which is again abnoraml in real life.
  • Datatype of “Aircraft” needs to be changed to factor as it contains only two values.

#Clean Abnormal values

FAA_Cleaned <- FAA_Nodup %>% 
  filter(duration > 40 | is.na(duration)) %>% 
  filter((speed_ground >= 30 && speed_ground <= 140) | is.na(speed_ground)) %>%  
  filter((speed_air >= 30 && speed_air <= 140) | is.na(speed_air)) %>%  
  filter(height >= 6 | is.na(height)) %>%  
  filter(distance < 6000 | is.na(distance))

We observe that the R-squared value increases with increase in number of variables.

However, the increase in R-square isn’t significant after adding more than 3 variables.

We now plot the same models using the Adjusted R-squared to check if we should add more variables or 3 variables can be enough.

#Calculating Adjusted R-square for each model and plotting these R-squared values against the number of variables.

plot(x=c(1,2,3,4,5,6),y=(c(summary(fit1)$adj.r.squared,summary(fit2)$adj.r.squared,summary(fit3)$adj.r.squared,summary(fit4)$adj.r.squared,summary(fit5)$adj.r.squared,summary(fit6)$adj.r.squared)),xlab = "Number of Variables",ylab = "Adjusted R-Squared Values")

We again note that the increase in R-square isn’t much after 3 predictor variables.

#Calculating AIC values for each model and plotting these values against the number of variables.

plot(x=c(1,2,3,4,5,6),y=(c(AIC(fit1),AIC(fit2),AIC(fit3),AIC(fit4),AIC(fit5),AIC(fit6))),xlab = "Number of Variables",ylab = "AIC Values")

We observe that the AIC values keeps on decreasing with increase in number of variables upto 5 variables and then increases slightly.

From past 3 plots, we can conclude that a good model can be built using 5 of the 6 variables from the data to determine the landing distance.

Thus, we will use ‘speed_ground’,‘aircraft’,‘height’,‘pitch’ and ‘duration’ variables to determine the Landing Distance in our Model.
